
# Tea Inside Bot S6.1

# Debug Mode
```
git clone https://github.com/TeaInside/tea-inside-bot-s6.1 teabot
cd teabot
make -j2
```

# Release Mode
```
git clone https://github.com/TeaInside/tea-inside-bot-s6.1 teabot
cd teabot
make release RELEASE_FLAG="-j2"
```

# Usage
```
cp teabot.conf.example teabot.conf
## Edit teabot.conf
./teabot
```
